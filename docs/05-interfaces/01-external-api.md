# 对外接口约束

## 1. Source of truth

对外 API 的签名、参数、错误码以公开头文件、NAPI/NDK/CJ 声明和 SDK 文档为准；本文不复制接口清单，避免与源码漂移。

## 2. 发布出口

SDK 头、NAPI、NDK、CJ 等公开出口必须经 `bundle.json` 的 `inner_kits` 显式登记。新增或修改公开 SDK 头签名、错误码、生命周期语义前需要升级确认。

## 3. ABI 约束

IPC code 数值是 ABI，不得修改已有值。

## 4. 权限与敏感信息

- 涉及系统接口、绑定、导入 PIN、可信关系管理时，必须按权限模型校验调用方身份。
- 不得明文输出设备标识、PIN、位置信息等敏感数据。
